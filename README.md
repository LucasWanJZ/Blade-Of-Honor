# JavaScript Fighting Game

Welcome to this simple JavaScript Fighting Game! This project is a browser-based fighting game developed using HTML, CSS, and JavaScript.

## Table of Contents

- [Demo](#demo)
- [Features](#features)
- [Game Controls](#game-controls)

## Demo

Check out the game in action: [https://gameproject-js-html-css.netlify.app/](#)

## Features

- Two-player fighting game
- Basic attack mechanics
- Jumping and movement controls
- Health bars for each player
- Different animations for different actions
- Background music and sound effects

## Game Controls

### Player 1

- Move Left: `A`
- Move Right: `D`
- Jump: `W`
- Attack: `R`

### Player 2

- Move Left: `ArrowLeft`
- Move Right: `ArrowRight`
- Jump: `ArrowUp`
- Attack: `ArrowDown`
